# clinicafisioapp

Sistema de gerenciamento de pacientes e agendamentos, feito com ASP.NET Core MVC + EF Core. Criado para ajudar minha esposa fisioterapeuta a organizar atendimentos diários.
** Projeto pessoal **

## Funcionalidades
✅ Cadastro de pacientes  
✅ Agendamentos com data/hora  
✅ Filtros e pesquisa  
🚧 Em desenvolvimento: visualização por calendário
